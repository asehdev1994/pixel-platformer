# Pixel Platformer

A 2D platformer built in Godot as part of my journey to develop stronger software engineering skills in an environment I genuinely enjoy.

While my professional background is in actuarial modelling and software development, this project gives me an opportunity to explore areas that are less common in my day-to-day work, including game architecture, event-driven programming, scene management, animation systems, and interactive user experiences.

Rather than aiming to build a complete commercial game, the focus of this repository is to practise writing clean, maintainable code while incrementally adding new gameplay mechanics and refactoring the project as it grows.

---

## Current Features

* Multiple playable levels
* Smooth level transitions with fade effects
* Collectible items and scoring system
* Enemy AI
* Environmental traps
* Player movement with double jump
* Enemy stomping mechanics
* Event-driven game flow using Godot signals

## Learning Goals

This project is being used to develop practical experience with:

* Scene-based architecture
* Object-oriented design
* Event-driven programming
* Code organisation and maintainability
* Version control with Git
* Incremental software development and refactoring

As the project evolves, additional systems such as menus, save/load functionality, character selection, level selection, and more advanced gameplay mechanics will be introduced while maintaining a clean and scalable architecture.
